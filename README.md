# 🍋 lemur_data – SQL Challenges & Solutions

Welcome to my personal repository where I document my journey through complex SQL problems. This project contains solutions to challenges from platforms like **DataLemur**, **Ace the Data Science Interview**, and **LinkedIn Daily SQL Puzzles**.

## 🚀 Why this repository?
The goal of this project is to showcase my ability to:
* Solve real-world data problems using advanced SQL techniques.
* Write clean, readable, and optimized queries.
* Think analytically about data aggregation and business logic.

## 📂 Project Structure
The repository is organized by platform or source:
* `/sql_interview_challenges` – Solutions from "Ace the Data Science Interview".
* `/linkedin_puzzles` – Daily SQL challenges found on LinkedIn.
* `/datalemur` – Specific tasks from the DataLemur platform.

## 🛠️ Key SQL Techniques Used
* **CTEs (Common Table Expressions)** for modular and readable code.
* **Complex Aggregations** (Aggregations on top of aggregations).
* **Window Functions** (`RANK`, `LEAD/LAG`, `ROW_NUMBER`).
* **Self-Joins** and complex filtering logic.
